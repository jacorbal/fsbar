FSBAR
=====

Report file system space usage in porcentage with a visual text bar.
The bar adapts to the screen width in case the terminal has a small
number of columns.


Example
-------

    $ fsbar
    [MOUNT POINT]           [USE %]
    /                        0.21%     [....................]
    /usr                    33.26%     [#######.............]
    /boot                   54.16%     [###########.........]
    /tmp                     0.01%     [....................]
    /var                    41.52%     [#########...........]
    /mnt/data0              48.57%     [##########..........]
    /home                    7.06%     [##..................]
